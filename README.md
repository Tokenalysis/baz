# Baz
[![hex.pm version](https://img.shields.io/hexpm/v/baz.svg?style=flat)](https://hex.pm/packages/baz)

A chain native hybrid bazaar engine

## Installation

Add the `baz` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:baz, "~> 0.0.1"}
  ]
end
```
